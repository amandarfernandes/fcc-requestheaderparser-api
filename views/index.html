Freecodecamp Backend Challenge #2

Request Header Parser Microservice

Objective

Build a full stack JavaScript app that is functionally similar to this: https://cryptic-ridge-9197.herokuapp.com/api/whoami/.

Here's the specific user story you should implement for this project:

  1. I can get the IP address, language and operating system for my browser.

Example usage:

https://fcc-mandycodestoo-reqheader-api.glitch.me/api/whoami

Example output:

{"ipaddress":"127.0.0.1","language":"en-US","software":"Macintosh; Intel Mac OS X 10_13_1"}
